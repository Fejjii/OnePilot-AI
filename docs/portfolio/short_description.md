# Short Portfolio Description

**OnePilot AI** is a production-style, multi-tenant AI operations platform for small businesses. It combines a company knowledge base (RAG with citations), a LangGraph agent for email/calendar/lead workflows, and mandatory human approval before any external action.

A live public demo runs on Vercel + Railway with one-click **Try the demo** access. On the public track, Gmail and Calendar use mock providers so reviewers can exercise the full approval path without sending real messages or creating real events.

**Stack:** FastAPI, LangGraph, PostgreSQL, Redis, Qdrant, Next.js, TanStack Query  
**Demo:** https://one-pilot-ai.vercel.app  
**Repo:** https://github.com/Fejjii/OnePilot-AI
